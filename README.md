# 💬 Chat Box - Ứng dụng nhắn tin thời gian thực

Một ứng dụng nhắn tin thời gian thực đa nền tảng được xây dựng bằng **Flutter** và **Firebase**, áp dụng **Clean Architecture** và quản lý trạng thái bằng **BLoC/Cubit**. 

## Tính năng nổi bật

* **Xác thực người dùng:** Đăng nhập, Đăng ký và Quên mật khẩu (Firebase Auth).
* **Nhắn tin Real-time:** Gửi/nhận tin nhắn văn bản với độ trễ cực thấp (Cloud Firestore).
* **Tương tác thông minh:** Vuốt để trả lời (Swipe to Reply), cập nhật trạng thái "Đã xem" (Seen).
* **Quản lý trạng thái trực quan:** Hiển thị người dùng Online (chấm xanh), huy hiệu đếm tin nhắn chưa đọc.
* **Đa ngôn ngữ (Localization):** Chuyển đổi linh hoạt giữa Tiếng Việt và Tiếng Anh không cần khởi động lại.
* **Cài đặt cá nhân:** Quản lý thông tin hồ sơ, thay đổi ảnh đại diện và tùy chỉnh ngôn ngữ.

## Công nghệ sử dụng

* **Framework:** [Flutter](https://flutter.dev/) (Dart)
* **Backend & Database:** Firebase Authentication, Cloud Firestore
* **State Management:** BLoC / Cubit (`flutter_bloc`)
* **Kiến trúc:** Clean Architecture
